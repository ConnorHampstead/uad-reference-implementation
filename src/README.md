# UAD Demo Application (Source)

This is a lightweight Node.js Express application used to demonstrate the **Unified Asset Delivery** security gates.

## Developer Setup

### Prerequisites
* Node.js v18+
* NPM

### Local Development
To run the app on your machine (outside the supply chain):

```bash
# 1. Install dependencies
npm install

# 2. Run the server
npm start
# Output: App listening on port 3000
```
